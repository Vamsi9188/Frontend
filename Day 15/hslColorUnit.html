Welcome to hsl() Color Units

hsl stands for HUE Saturation Lightness

h(hue): value in degree(0-359)

s(saturaion):determines the vibrancy of the color or how grayed.

l(lightness):how light or dark the color should be.

hsl() Color unit takes the values in numbers

First values in 0-359 range in degrees

Second and Third values should be in range 0-100

Either in number or in Percentages
