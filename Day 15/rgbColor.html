Welcome to RGB() Color Units

RGB Means RED GREEN BLUE

First value is in 0-255 range of Red Color

Second value is in 0-255 range of Green Color

Third value is in 0-255 range of Blue Color

If three values are 0 then it is black Color

If three values are 255 the it is white color

The values of opacity is between 0 and 1

The values are taken in terms of percentages 0%-100%
